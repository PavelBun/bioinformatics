# Домашнее задание 2: изучение представленности гена TP53 в таксонах

## 1. Название выбранного гена (Gene Symbol)

**TP53**

Выбранный ген кодирует белок p53 — опухолевый супрессор, участвующий в регуляции клеточного цикла, ответа на повреждение ДНК и запуске апоптоза.

## 2. Параметры BLAST, использованные для поиска гомологичных генов

Для поиска гомологичных последовательностей использовался инструмент **NCBI BLASTP** по белковой последовательности p53 человека.

| Параметр | Значение |
|---|---|
| Программа | BLASTP |
| База данных | ClusteredNR |
| Исходная последовательность | Homo sapiens p53 protein |
| Accession исходной последовательности | NP_000537.3 |
| Тип молекулы | amino acid / protein |
| Длина query | 260 aa |
| Матрица замен | BLOSUM62, по умолчанию |
| E-value threshold | 10, по умолчанию |
| Фильтр low complexity regions | включён, по умолчанию |

Критерии отбора последовательностей: выбирались гомологичные белки p53/TP53 не из Homo sapiens, с высоким query coverage, низким E-value и достаточным процентом идентичности.

## 6. Краткий анализ консервативности выравнивания

Множественное выравнивание показало, что белок p53 является высоко консервативным среди выбранных видов млекопитающих. В выравнивании присутствует большое количество позиций, отмеченных символом `*`, что соответствует полностью совпадающим аминокислотам во всех последовательностях.

Наиболее высокая степень консервативности наблюдается в центральной части белка p53. Этот участок соответствует ДНК-связывающему домену, который играет ключевую роль в функции p53 как транскрипционного фактора и опухолевого супрессора. Сохранность данного домена указывает на его функциональную значимость.

N-концевая часть белка также содержит консервативные мотивы, хотя в ней наблюдается больше вариабельности по сравнению с центральным доменом. В отдельных участках выравнивания присутствуют вставки и делеции, что отражает эволюционные различия между видами. Такие изменения чаще встречаются в менее консервативных областях белка.

Таким образом, результаты множественного выравнивания подтверждают, что p53 сохраняет основные функционально важные участки в ходе эволюции. Это согласуется с его ролью в регуляции клеточного цикла, ответе на повреждение ДНК и подавлении опухолевого роста.

## 6. Объединяющий таксон

**Латинское название:** Mammalia  
**Русское название:** Млекопитающие

Все выбранные виды относятся к классу **Mammalia**. Это группа позвоночных животных, для которых характерны наличие молочных желёз, волосяного покрова, развитая нервная система и теплокровность. В выборку вошли представители разных групп млекопитающих: приматы, грызуны, китообразные, хищные и шерстокрылы. Наличие гомологичных последовательностей TP53 у этих организмов указывает на древнее происхождение и важную консервативную функцию белка p53 у млекопитающих.

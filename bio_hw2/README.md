# TP53 homology analysis

Этот репозиторий содержит материалы для домашнего задания 2 по анализу представленности гена TP53 в таксонах.

## Содержимое

- `report.md` - текст отчёта.
- `sequences.fasta` - исходные белковые последовательности p53/TP53, использованные для выравнивания.
- `clustalo-I20260427-120241-0518-87793837-p1m.aln-clustal_num` - множественное выравнивание в формате Clustal.
- `homologs_table.tsv` - таблица видов и accession-номеров.
- `photo.png` - фото выравнивания.
